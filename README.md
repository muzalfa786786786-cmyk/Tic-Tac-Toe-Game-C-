# ❌⭕ Tic Tac Toe Game (C++)
# 📌 Task Objective

The objective of this project is to develop a console-based Tic Tac Toe game using C++ programming language that allows users to play either in single-player or two-player mode. The system provides an interactive gameplay experience by implementing game logic through programming constructs.

# 🛠️ Tasks Performed

Developed a console-based game interface

Implemented single-player and two-player game modes

Provided symbol selection option (‘X’ or ‘O’) for players

Created game board using 2D Array

Implemented player turn management system

Applied game winning logic for:

Horizontal matches

Vertical matches

Diagonal matches

Displayed game result such as:

Player Win

Player Loss

Match Draw

Implemented replay option for users to play again

# 📂 Deliverables

Functional Tic Tac Toe Game

Interactive Console-Based Interface

Game Result Display System

Replay Functionality

# 📚 Technologies Used

C++ Programming Language

Arrays

Loops

Functions

Conditional Statements

String Handling
